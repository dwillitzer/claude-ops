## ISSUE PROTOCOL

### Adding Issues
```
1. Assign ID: ISSUE-[YYYY-MM-DD]-[SEQ]
2. Set severity: Critical / High / Normal
3. Identify blocking impact
4. Specify what's needed to resolve
5. Escalate if Critical
```

### Resolving Issues
```
1. Document resolution
2. Move to RESOLVED section
3. Update activeContext.md
4. Keep in RESOLVED for 5 sessions, then archive
```

### Escalation
```
Critical issues auto-escalate to Business Director
If unresolved > 2 sessions, escalate to human
```
